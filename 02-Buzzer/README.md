# Arduino Buzzer

A simple Arduino project that controls a buzzer using digital pin 8.

## Components
- Arduino Uno
- Buzzer

## Connections
- Buzzer (+) → Arduino Digital Pin 8
- Buzzer (-) → Arduino GND

## Working
The buzzer turns ON for 500 milliseconds and OFF for 500 milliseconds continuously.

## Technologies
- Arduino Uno
- Arduino C/C++
- Tinkercad
