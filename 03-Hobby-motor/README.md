# 03 - Hobby Motor Control with Arduino

## 📌 Project Description

This project demonstrates how to control a hobby motor using an Arduino Uno.

The motor is connected to the Arduino using a transistor-based switching circuit. A resistor is used to control the transistor input, allowing the Arduino to turn the motor ON and OFF.

## 🛠️ Components Required

- Arduino Uno
- Hobby Motor / DC Motor
- NPN Transistor
- 1kΩ Resistor
- Diode (1N4007)
- Breadboard
- Jumper Wires
- External Power Supply (if required)

## 🔌 Circuit Connections

The circuit is connected as shown in the `01-Circuit` folder.

### Connections

- Arduino Digital Pin 8 → Resistor → Transistor Base
- Transistor Emitter → Arduino GND
- Motor → Transistor Collector
- Motor → Power Supply
- Diode → Connected across the motor terminals for protection

> **Note:** A transistor is used as a switch because an Arduino GPIO pin should not directly drive a motor. The diode helps protect the circuit from voltage spikes generated by the motor.

## 💻 Arduino Code

The Arduino code is available in the `02-Code` folder.

The program controls the motor using Digital Pin 8.

When Pin 8 is set to HIGH, the motor turns ON.

When Pin 8 is set to LOW, the motor turns OFF.

## 🎥 Project Demo

The working demonstration video is available in the `03-Demo` folder.

## ⚙️ Working Principle

The Arduino sends a digital HIGH or LOW signal through the resistor to control the transistor.

### Motor ON

When Digital Pin 8 is HIGH:

Arduino → Resistor → Transistor turns ON → Current flows through the motor → Motor rotates.

### Motor OFF

When Digital Pin 8 is LOW:

The transistor turns OFF → Current stops flowing through the motor → Motor stops.
